# Symbolic-oriented Advertising Image Prompt 中文说明

## 共享实验指令
你正在为消费者广告实验生成受控刺激材料。

请把提供的白底商品图作为准确的源商品。保持商品身份不变，保留商品的核心形状、材质、颜色、包装、logo、可见设计和物理结构。

不要虚构商品功能、成分、认证、奖项、功效声明、口号、评分、价格、二维码、徽章、图表、箭头或品牌资产。

不要在图中生成新的可读文字。源商品图中原本已有的文字、logo 或包装信息可以保留，但不要新增广告文案，也不要重写包装文字。请保留干净留白，方便后续用受控模板叠加文字。

避免截图感、横幅模板感、低质量拼贴、表情包风格、杂乱电商 mockup、畸形手部、破损包装、乱码字符和不真实的商品几何结构。

## 生成 Symbolic-oriented 广告图

主要说服路径：象征意义、品牌人格、自我表达、身份暗示、视觉修辞和 mood。不要把功能解释、密集产品信息或具体使用演示作为主要说服路径。

### 广告机制
图像应该通过让商品具有意义来塑造观看者对商品或品牌的态度：例如地位、关怀、品味、亲密、向往、庆祝、专业、自然、活力、个性，或其他与品牌兼容的象征意义。审美和情绪是表达象征意义的执行手段，而不是这个分类本身。

### 象征推断指令
- 从商品标题、品牌、类目、价格水平和购买场景中推断一个主导象征领域。
- 从以下受控词汇中选择一个主导品牌人格，并让整张图围绕它展开：glamorous, rugged, healthy, fun, sincere, sophisticated, warm, futuristic, natural, playful, premium, refined。
- 如果商品标题和品牌没有强烈暗示具体人格，则按品类惯例推断：护肤或美妆倾向 sincere、natural 或 refined；电子产品倾向 futuristic、premium 或 sophisticated；食品饮料倾向 warm、fun 或 healthy；运动或户外产品倾向 rugged、energetic 或 playful；珠宝或奢侈品倾向 glamorous、sophisticated、premium 或 refined。
- 只能通过视觉处理隐含表达所选象征领域和人格。不要把人格词、口号或 mood 标签写进图像。

### 视觉方向
- 让商品成为画面的视觉锚点，但把它呈现为象征物，而不是信息说明物。
- 使用光线、色彩、背景材质、景深、阴影、反射、构图和视觉隐喻来传达所选象征意义。
- 有意识地使用视觉修辞：对称可以表达 prestige，柔和光晕可以表达 intimacy 或 care，强对比可以表达 luxury 或 aspiration，有机材质可以表达 naturalness，精确几何可以表达 expertise 或 futurism，也可以使用其他与品类兼容的象征形式。
- 有意识地控制色彩复杂度：expressive、playful、fun、glamorous 或 celebratory 意义可使用更丰富、分层的色彩；premium、sophisticated、refined、sincere 或 calm 意义应使用更简洁、单色或克制的色彩。
- 通过象征联想、品牌人格和自我表达让商品变得有吸引力，而不是通过明确产品信息或使用演示。
- 图像应在 2 秒内唤起感受和清晰品牌印象，不依赖观看者阅读文字。
- 图像可以类似高级品牌视觉、象征性产品肖像、视觉隐喻、奢侈品静物或高端社交媒体 campaign 图。

### 严格边界
- 不要把详细具体使用场景作为画面核心。
- 不要使用功能标签、对比图表、密集信息设计、产品特征说明或促销导向版式。
- 不要让产品功能解释或具体使用演示成为主导视觉印象。
- 不要新增文字、口号、假标签、假徽章、评分或无支撑声明。
- 不要让商品变得难以识别、太小、过度抽象或隐藏在氛围中。
- 除非象征意义和品类明确需要，不要使用人脸或身份线索。
- 不要改变商品形状、颜色、logo、包装或核心外观。

### 验证标准
观看者应能在“这张图让我清楚感受到品牌的象征意义”和“这张图传达了鲜明的品牌人格或自我表达 mood”上给高分。象征影响应来自氛围、构图、视觉隐喻和品牌兼容的审美，而不是广告文案、产品特征说明或具体使用演示。

### 输出目标
生成一张广告图，首要目标是象征意义、品牌人格感知、自我表达、欲望和广告态度。审美吸引力应该服务于象征说服，而不是替代象征说服。
