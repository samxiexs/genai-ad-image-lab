# Symbolic-oriented Function v2 Advertising Image Prompt 中文说明

## 共享实验指令
你正在为消费者广告实验生成受控刺激材料。

请把提供的白底商品图作为准确的源商品。保持商品身份不变，保留商品的核心形状、材质、颜色、包装、logo、可见设计和物理结构。

不要虚构商品功能、成分、认证、奖项、功效声明、口号、评分、价格、二维码、徽章、图表、箭头或品牌资产。

不要在图中生成新的可读文字。源商品图中原本已有的文字、logo 或包装信息可以保留，但不要新增广告文案，也不要重写包装文字。请保留干净留白，方便后续用受控模板叠加文字。

避免截图感、横幅模板感、低质量拼贴、表情包风格、杂乱电商 mockup、畸形手部、破损包装、乱码字符和不真实的商品几何结构。

## 生成 Symbolic-oriented 广告图

主要说服路径：象征意义、品牌人格、自我表达、身份暗示和视觉修辞。图像必须传达商品“意味着什么”或“拥有者表达了什么”，而不是商品做什么、如何工作或有哪些可见功能。

### 广告机制
图像应该通过让商品成为有意义的符号来塑造观看者对商品或品牌的态度：例如地位、关怀、品味、亲密、向往、庆祝、专业、自然、活力、个性、归属、转变、自信，或其他与品牌兼容的象征意义。审美和 mood 是表达象征意义的执行手段，而不是分类本身。

### 象征推断指令
- 从商品标题、品牌、类目、价格水平和购买场景中推断一个主导象征领域。
- 从以下受控词汇中选择一个主导品牌人格，并让整张图围绕它展开：glamorous, rugged, healthy, fun, sincere, sophisticated, warm, futuristic, natural, playful, premium, refined。
- 如果商品标题和品牌没有强烈暗示具体人格，则按品类惯例推断：护肤或美妆倾向 sincere、natural 或 refined；电子产品倾向 futuristic、premium 或 sophisticated；食品饮料倾向 warm、fun 或 healthy；运动或户外产品倾向 rugged、energetic 或 playful；珠宝或奢侈品倾向 glamorous、sophisticated、premium 或 refined。
- 只能通过视觉处理隐含表达所选象征领域和人格。不要把人格词、口号或 mood 标签写进图像。

### 视觉方向
- 让商品保持可识别并成为视觉锚点，但把它呈现为身份符号或意义载体，而不是信息说明物。
- 使用氛围、构图、色彩、光线、阴影、材质、景深、反射、尺度、留白和视觉隐喻来传达所选象征意义。
- 必须包含清晰的象征视觉装置或隐喻。例如：对称表达 prestige，地平线或上升感表达 aspiration，柔和光晕表达 care 或 intimacy，有机形态表达 naturalness，精确几何表达 futurism 或 expertise，庆祝性光线表达 joy，粗粝地形材质表达 resilience，或其他与品类兼容的象征装置。
- 图像可以具有氛围感、编辑感、电影感、可信的超现实感或 campaign 感，只要商品仍然可识别且象征意义与品牌兼容。
- 单纯高级好看不够。只展示材质、工艺、包装或陈列的干净奢侈品静物，如果没有清晰象征领域，则视为失败。
- 保留干净留白，方便后续统一叠加受控文字。

### 严格边界
- 不要把产品功能、机制、安装、适配、兼容、耐用、收纳、新鲜度、护理或易用性作为主要视觉信息。
- 不要使用部件布局、局部细节诊断区、中性演示道具、尺度参照、产品特征说明、对比图表、标签、箭头或产品信息设计。
- 不要把具体日常使用场景作为画面核心。
- 不要让可见功能证据或实际使用 affordance 成为主导视觉印象。
- 不要新增文字、口号、假标签、假徽章、评分或无支撑声明。
- 不要让商品变得难以识别、太小、过度抽象或隐藏在氛围中。
- 除非象征意义和品类明确需要，不要使用人脸或身份线索。
- 不要改变商品形状、颜色、logo、包装或核心外观。

### 验证标准
观看者应能在“这张图让我清楚感受到品牌的象征意义”和“这张图传达了鲜明的品牌人格或自我表达 mood”上给高分。图像应通过 3 秒意义测试：观看者应能推断商品代表什么或拥有它表达什么，同时不会主要学习商品做什么，也不会主要想象一个具体使用场景。

### 输出目标
生成一张广告图，首要目标是象征意义、品牌人格感知、自我表达、欲望和广告态度。审美吸引力应该服务于象征说服，而不是替代象征说服。
