# Symbolic-oriented Advertising Image Prompt v12 中文说明

你正在为消费者广告实验生成受控刺激材料。

使用输入的白底商品图作为 exact source product。保持商品身份不变，包括核心形状、材质、颜色、包装、logo、可见设计和物理结构。

不要虚构未被支持的商品功能、成分、认证、奖项、claims、广告语、评分、价格、二维码、徽章、图表、箭头或品牌资产。

不要在图像中生成新的可读文字。源商品上已有文字、logo 或包装信息可以保留，但不要新增广告文案或改写包装文字。为后续受控文字叠加保留干净留白。

避免截图感、横幅模板感、低质量拼贴、meme 风格、杂乱电商 mockup、变形手部、破损包装、不可读乱码和不真实商品几何。

生成基于 Park, Jaworski, and MacInnis (1986) 的 Symbolic-oriented 广告图，并让 SYMBOLIC brand concept 成为主导构念。

Brand concept definition：
Park et al. 将 brand concept 定义为基于消费者需求而选择的品牌意义。本 prompt 表达 symbolic brand concept。Symbolic 指商品与自我提升、角色位置、群体归属、ego-identification 或理想自我形象等内部需求相连。

Primary route of persuasion：
主导路径是 symbolic meaning、self-expression、identity signaling、role meaning、group meaning 和 visual rhetoric。图像必须传达商品在 symbolic brand concept 下“意味着什么/对拥有者说明了什么”，而不是商品做什么、如何工作，或它有哪些可见特征。

Advertising mechanism：
图像应通过把商品变成有意义的符号来塑造态度：status、care、taste、intimacy、aspiration、celebration、expertise、naturalness、vitality、individuality、belonging、transformation、confidence 或其他品牌兼容意义。美感和氛围是执行工具，不是分类本身。

Symbolic reasoning instruction：
- 结合 Park symbolic 定义以及商品标题、品牌、品类、价格层级和购买场景，推断一个主导 symbolic territory。
- 选择一个主导且品牌兼容的 symbolic territory，并让整张图围绕它展开。
- 如果标题和品牌没有强烈指向，就按品类惯例推断：护肤美妆偏 sincere、natural 或 refined；电子产品偏 futuristic、premium 或 sophisticated；食品饮料偏 warm、fun 或 healthy；运动户外偏 rugged、energetic 或 playful；珠宝奢侈品偏 glamorous、sophisticated、premium 或 refined。
- 通过视觉处理隐性表达 symbolic territory，不要把 personality word、slogan 或 mood label 直接写成图片文字。

Visual direction：
- 保持商品作为可识别的视觉锚点，但要把它呈现为 identity symbol 或 meaning carrier，而不是信息说明物体。
- 使用 atmosphere、composition、color、light、shadow、texture、depth、reflection、scale、negative space 和 visual metaphor 来传达 symbolic meaning。
- 包含一个清晰 symbolic visual device 或 metaphor，例如：symmetry 表 prestige，horizon 或 elevation 表 aspiration，soft radiance 表 care 或 intimacy，organic forms 表 naturalness，precise geometry 表 futurism 或 expertise，celebratory light 表 joy，rugged terrain texture 表 resilience。
- 图像可以是 atmospheric、editorial、cinematic、surreal-but-plausible 或 campaign-like，只要商品可识别且意义与品牌兼容。
- 单纯 premium beauty 不够。一个只展示材质、工艺、包装或陈列的 clean luxury still life，如果没有明确 symbolic territory，就算失败。
- 视觉内容必须与 Park symbolic 定义、商品标题、品类、价格层级、促销语境和源图保持语义一致。
- 为后续受控文字叠加保留干净留白。

Strict boundaries：
- 不要把商品功能、机制、setup、fit、compatibility、durability、storage、freshness、care 或 ease of use 做成主视觉信息。
- 不要使用 component layout、close-detail diagnostic area、中性演示道具、尺度参照、feature exposition、comparison chart、label、arrow 或 product-information design。
- 不要把具体 everyday usage scene 做成主焦点。
- 不要让 functional evidence 或 practical use affordance 成为主导视觉印象。
- 不要加入生成文字、广告语、假标签、假徽章、评分或未被支持的 claims。
- 不要让商品难以识别、过小、过度抽象或被氛围遮蔽。
- 除非 symbolic meaning 和产品品类明确要求，否则不要使用正面人脸或强身份线索。
- 不要改变商品的形状、颜色、logo、包装或核心外观。

Verification standard：
观看者应能在 “This image gives me a clear sense of the product's symbolic meaning” 和 “This image communicates a distinctive self-expressive or identity-relevant meaning” 上给高分。图像应通过 3 秒 symbolic concept test：观看者应能快速推断商品象征什么或拥有它表达什么，而不是主要学习它做什么或联想到具体使用情境。
