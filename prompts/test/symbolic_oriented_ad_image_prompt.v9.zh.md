# Symbolic-oriented Advertising Image Prompt v9 中文说明

你正在为消费者广告实验生成受控刺激材料。

使用输入的白底商品图作为 exact source product。保持商品身份不变，包括可见形状、材质、颜色、包装、logo、设计细节和物理结构。

不要虚构输入未支持的商品功能、成分、材质、认证、奖项、claims、广告语、评分、价格、二维码、徽章、图表、箭头或品牌资产。

不要在图像中生成新的可读文字。源商品上已有的文字、logo 或包装信息可以保留，但不要新增广告文案或改写包装文字。

Image-quality controls：
- 生成一张连贯、精致的商业广告图。
- 商品必须是主视觉，清晰、光线充分、比例合理、易于查看。
- 避免截图感、横幅模板感、低质量拼贴、破损包装、不可读乱码、不真实商品几何和物理上不合理的互动。
- 不要只留下白底 packshot；在保持商品身份的同时增加广告图价值。

Brand concept definition：
依据 Park, Jaworski, and MacInnis (1986)，这里表达 symbolic brand concept。Symbolic 指商品与自我提升、角色位置、群体归属、ego-identification 或理想自我形象等内部需求相连。

Primary route of persuasion：
图像应让观看者理解商品的象征意义。观看者应能推断拥有、展示或关联该商品表达了怎样的 self-image、role、group membership、ego-identification 或 desired identity。不要让功能解释或感官氛围成为主要说服路径。

Concept-image linkage：
- 从源图和元数据推断一个合理 symbolic meaning。
- 把该意义转化为 identity、taste、belonging、aspiration、expertise、care、status 或 self-expression 的宽泛视觉线索。
- 只有在支持 symbolic meaning 且不违背商品事实时，才使用氛围、场景、构图、光线、尺度、对比、留白或克制的视觉隐喻。
- 人物、道具或环境只有在表达 symbolic meaning 时才可出现；避免无依据刻板印象、阶层 claims 或人口统计假设。
- 商品必须作为意义载体保持可识别，不能沦为小装饰物。

Verification standard：
图像应通过 3 秒 symbolic concept test：观看者应快速推断商品象征什么或拥有它表达什么，而不是主要学习功能或只感到感官刺激。
