# Context-oriented Function v2 Advertising Image Prompt 中文说明

## 共享实验指令
你正在为消费者广告实验生成受控刺激材料。

请把提供的白底商品图作为准确的源商品。保持商品身份不变，保留商品的核心形状、材质、颜色、包装、logo、可见设计和物理结构。

不要虚构商品功能、成分、认证、奖项、功效声明、口号、评分、价格、二维码、徽章、图表、箭头或品牌资产。

不要在图中生成新的可读文字。源商品图中原本已有的文字、logo 或包装信息可以保留，但不要新增广告文案，也不要重写包装文字。请保留干净留白，方便后续用受控模板叠加文字。

避免截图感、横幅模板感、低质量拼贴、表情包风格、杂乱电商 mockup、畸形手部、破损包装、乱码字符和不真实的商品几何结构。

## 生成 Context-oriented 广告图

主要说服路径：具体使用想象和自我代入。图像应该让观看者想象自己在真实日常情境中自然接触、手持、穿戴、摆放、赠送、消费或使用该商品。不要把产品功能解释或抽象象征氛围作为主要说服路径。

### 广告机制
图像应该回答消费者会在什么地点、什么时候、以什么方式自然使用或接触该商品。商品应该嵌入可信时刻，而不是被呈现为棚拍证据展示或象征品牌图标。

### 商品元数据

- 品牌：{creative_id_brand}
- 类目：{level_one_category_name}
- 价格：{creative_id_price}
- 促销：{creative_id_promotion}

### 使用情境推断指令
- 从商品标题、类目、目标用户、价格水平和购买语境中推断最可能的单一使用时机。
- 在整张图中坚持一个具体时机，例如晨间流程、通勤、运动、送礼、用餐、美妆护理、家居清洁、宠物照护、办公使用、旅行、户外活动，或其他同样具体的情境。
- 不要在一张图中混合多个使用时机。

### 视觉方向
- 把商品放在可信、符合品类的使用语境中，并使用真实环境线索。
- 商品在场景中仍需清楚可见且可识别，但不要像目录静物一样孤立展示。
- 偏向自然的社交媒体原生构图：第一人称、手持、局部身体、越肩视角、背影视角、桌面使用中、包中使用、货架使用中、办公桌使用中、厨房使用中，或其他自然消费者视角。
- 除非品类明确需要表情或身份，否则避免完整正面人脸。如果出现人脸，应保持局部、自然角度或轻微虚焦。
- 人与物的互动必须符合物理现实，包括比例、握持、姿势和摆放。
- 背景和前景物体只能服务于所选使用时机。
- 保留干净留白，方便后续统一叠加受控文字。

### 严格边界
- 不要把图像做成功能解释、部件布局、细节诊断图或中性棚拍台上的产品演示。
- 不要让图像主要变成抽象 mood、地位象征、象征意义或品牌人格。
- 不要把商品缩小成背景中的装饰物。
- 不要新增文字、口号、假标签、假徽章、评分或无支撑声明。
- 不要使用与品类无关的泛化背景。
- 不要创造不真实的手、畸形身体、错误比例或不可能的互动。
- 不要改变商品形状、颜色、logo、包装或物理结构。

### 验证标准
观看者应能在“这张图帮助我想象自己使用这个产品”和“这张图展示了真实使用情境”上给高分。图像应通过 3 秒使用测试：观看者不读文字也能推断产品会在何处、何时、如何被使用，同时不会主要把它理解为功能解释图或象征 mood 图。

### 输出目标
生成一张广告图，优化目标是使用想象、自我代入、真实使用情境、产品-场景匹配和互动参与。
