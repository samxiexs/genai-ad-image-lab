# Context-oriented Advertising Image Prompt 中文说明

## 共享实验指令
你正在为消费者广告实验生成受控刺激材料。

请把提供的白底商品图作为准确的源商品。保持商品身份不变，保留商品的核心形状、材质、颜色、包装、logo、可见设计和物理结构。

不要虚构商品功能、成分、认证、奖项、功效声明、口号、评分、价格、二维码、徽章、图表、箭头或品牌资产。

不要在图中生成新的可读文字。源商品图中原本已有的文字、logo 或包装信息可以保留，但不要新增广告文案，也不要重写包装文字。请保留干净留白，方便后续用受控模板叠加文字。

避免截图感、横幅模板感、低质量拼贴、表情包风格、杂乱电商 mockup、畸形手部、破损包装、乱码字符和不真实的商品几何结构。

## 生成 Context-oriented 广告图

主要说服路径：使用想象和自我代入。不要把产品信息标签或抽象情绪象征作为主要说服路径。

### 广告机制
图像应该帮助观看者想象自己在真实日常情境中使用、拥有、手持、穿戴、赠送、消费或摆放该商品。

### 商品元数据
- 标题：{ori_title}
- 品牌：{creative_id_brand}
- 类目：{level_one_category_name}
- 价格：{creative_id_price}
- 促销：{creative_id_promotion}

### 视觉方向
- 把商品放在可信、符合品类的使用语境中。
- 场景应该回答：消费者会在什么地点、什么时候、以什么方式自然使用或接触该商品？
- 根据具体商品标题和品类推断最可能的单一使用时机，并在整个画面中坚持这一个时机。不要在一张图中混合多个使用时机。
- 将场景落到具体使用时机，例如晨间流程、通勤、运动、送礼、用餐、美妆护理、家居清洁、宠物照护、办公使用、户外活动，或其他同样具体的情境。
- 商品在场景中仍需清楚可见且可识别。
- 使用与品类、使用时机和消费者情境匹配的环境线索。
- 偏向自然的社交媒体原生构图，而不是摆拍式目录图。
- 优先使用第一人称、手持、局部身体、越肩视角或背影视角，让消费者隐含成为商品使用者。
- 除非品类明确需要表情或身份，否则避免完整正面人脸。
- 如果出现人脸，应保持局部、自然角度或轻微虚焦。不要生成高度真实的正面 AI 人脸，以免触发真实性顾虑。
- 人与物的互动必须符合物理现实，包括比例、握持、姿势和摆放。
- 背景环境和前景物体应与商品标题和品类语义一致。

### 严格边界
- 不要让图像主要变成抽象情绪或象征性 mood。
- 不要让干净产品展示或抽象品牌氛围成为主导视觉印象。
- 不要把商品缩小成背景中的装饰物。
- 不要新增文字、口号、假标签、假徽章、评分或无支撑声明。
- 不要使用与品类无关的泛化背景。
- 不要创造不真实的手、畸形身体、错误比例或不可能的互动。
- 不要改变商品形状、颜色、logo、包装或物理结构。

### 验证标准
观看者应能在“这张图帮助我想象自己使用这个产品”和“这张图展示了真实使用情境”上给高分。使用时机应足够具体，使观看者能推断该商品会在何处、何时、如何被使用，同时不会主要把它理解为干净产品展示图或抽象情绪图。

### 输出目标
生成一张广告图，优化目标是使用想象、自我代入、真实使用情境、产品-场景匹配和互动参与。
