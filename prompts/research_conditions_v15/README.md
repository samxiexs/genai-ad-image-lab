# Research Conditions V15

This folder stores the copied `v15` definition-only research-condition prompt family.

It is copied from `prompts/research_conditions_v14/` so `definition-only-v15` can be run as the successor version while preserving the same orientation definitions.

Runtime version names:

- `definition-only-v15`

Note: the historical bare `--prompt-version v15` prompt family is defined separately under `prompts/test/` and is also copied from v14. Use `definition-only-v15` for this folderized definition-only version.

Folder layout:

- `product_oriented/`
- `function_oriented/`
- `symbolic_oriented/`
- `experiential_oriented/`

Each orientation folder contains:

- `definition-only.txt`

Design notes:

- `Product-oriented` and `Function-oriented` intentionally use the same functional definition and differ only in orientation label.
- `definition-only-v15` keeps the copied v14 study definitions and uses a single explanation paragraph.
- Functional explanations emphasize functional performance, problem-solving specialization/generalization, and performance-related products.
- Symbolic explanations emphasize self-identification, symbolic reference groups, exclusivity, and lifestyle image extension.
- Experiential explanations emphasize sensory satisfaction/cognitive stimulation, experiential and fantasy aspects of consumption, related stimulation, and links to other experiential products.
