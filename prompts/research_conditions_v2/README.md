# Research Conditions V2

This folder stores the independent refactor of the four research-condition families.

It exists so the original definition-based prompt files under `prompts/test/` and `prompts/aliases/` remain unchanged.

Runtime version names:

- `definition-only-v2`
- `definition-control-v2`
- `visual-control-v2`
- `definition-genprompt-v2`
- `definition-control-genprompt-v2`
- `genprompt-control-v2`

These `-v2` versions are mapped directly by `scripts/generate_images/generate_from_csv.py` to the files in this folder.
