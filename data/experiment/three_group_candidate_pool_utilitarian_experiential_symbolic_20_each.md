# 三类商品候选池（Utilitarian / Experiential Hedonic / Symbolic Hedonic）

这版按照你最新确认的定义来分：

1. `strict_utilitarian`
2. `experiential_affective_hedonic`
3. `symbolic_aesthetic_hedonic`

这里的关键变化是：

- 不再使用 `mixed`
- `hedonic` 明确拆成两类
- 第二类更偏 `experiential / sensory / affective / pleasure-oriented`
- 第三类更偏 `aesthetic / symbolic / self-expression / public display`

## 我对这版的判断

这一版比上一版更贴你现在的理论框架：

- `utilitarian` 还是“问题解决型”
- `experiential hedonic` 更像“闻起来/吃起来/喝起来/玩起来/氛围上如何”
- `symbolic hedonic` 更像“我看起来像谁 / 属于什么风格 / 呈现什么生活方式”

## 我会特别提醒的边界项

这些不是错放，但如果你后面想继续收紧，可以优先考虑替换：

- `experiential hedonic` 里的高奢香水：仍然会带一点品牌象征性
- `鲜花` 和 `香薰礼盒`：会带一点送礼语境
- `symbolic hedonic` 里的部分平底鞋或通勤包：会带一点实用性，但整体仍以审美表达为主
