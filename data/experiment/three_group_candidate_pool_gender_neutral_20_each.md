# 三类商品候选池（性别中性优先版）

这版是在上一轮基础上，额外加入了一个硬约束：

- 商品尽可能 `gender-neutral`

我的处理原则是：

1. 尽量剔除标题里直接出现 `女 / 男 / 女士 / 男士 / 女友 / 新娘 / 宝宝 / 妈妈` 等强性别或强角色词的商品
2. 即使标题没有性别词，也尽量少选默认被理解为“强女性时尚品”或“强男性配件”的商品
3. 对 `symbolic-aesthetic hedonic`，优先保留更中性的公开展示类商品：
   - 太阳镜
   - 运动鞋
   - 手表
   - 行李箱
   - 家居审美摆件
4. 对 `experiential-affective hedonic`，优先保留：
   - 茶、酒、零食、香氛、积木、小说这类更少性别定向的商品

## 使用建议

如果你现在最在意“去掉性别混淆”，我会建议优先从下面这些方向挑：

- `strict_utilitarian`
  - 血压计、雾化机、打印机、洗烘套装、抽纸、拖把
- `experiential_affective_hedonic`
  - 茶、鸡尾酒、起泡酒、零食、香薰、小说、乐高
- `symbolic_aesthetic_hedonic`
  - 手表、太阳镜、运动鞋、行李箱、设计花瓶

## 我对这版的提醒

- `experiential` 里如果你想进一步去掉性别线索，最该优先删的是高奢香水
- `symbolic` 里如果你想进一步去掉时尚性别偏向，最稳的是继续往：
  - 太阳镜
  - 运动鞋
  - 行李箱
  - 中性手表
  - 家居设计摆件
  这些品类收缩
